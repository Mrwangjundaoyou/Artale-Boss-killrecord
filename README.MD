软件使用很简单

step1：选择将要击杀的Boss

step2：找到Boss后填写线路，击杀Boss后点击记录击杀时间

step3：等预计刷新状态变为刷新时回去寻找，如果回去仍然没刷新，可以点击复核情况下的按钮。

tips：

1.一般在大概率刷新的时候过去都能找到。如果出现软件卡住的情况，随意点击一个按钮即可。

2.插件已经更新保存和读取功能，保存时在软件相同目录下会生成两个文件，一个是txt，一个是ini
其中txt会记录击杀的boss名称，线路，时间，和预测刷新时间，目的是方便记录不同boss的击杀时间。
ini则是上一次保存的界面配置文件，点击以后会自动恢复到上一次的记录界面，方便各位使用。
